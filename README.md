# DevOps Onboarding Project

## Purpose
This repository simulates a small production-like system used to learn
and demonstrate DevOps practices: CI/CD, Docker, Terraform, and AWS-style infrastructure.

## System Overview
- Containerized API exposing /health and /metrics
- CI pipeline for testing and image builds
- Infrastructure defined with Terraform
- Docker-based runtime on EC2
- Basic observability and rollback strategy
